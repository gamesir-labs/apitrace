# api

## 作用

`src/api/` 提供仓库级的公共入口头文件和最薄的共享 API 聚合层。

当前主要包含：

- umbrella header
- 版本字符串入口

## 负责的内容

- 把各模块公共接口聚合为稳定的 `apitrace/...` 入口
- 暴露最小公共版本信息
- 作为外部使用方看到的头文件装配层

## 不负责的内容

- 不实现 capture
- 不实现 retrace
- 不定义 bundle 磁盘格式细节
- 不承载 D3D11 / D3D12 / Metal 的具体逻辑

## 当前实现边界

- `include/` 里主要是 API 汇总入口
- `src/` 里只保留极少量共享实现

## 当前阶段说明

这里仍然是架构骨架，不追求稳定 ABI，也不承诺接口已经定型。
