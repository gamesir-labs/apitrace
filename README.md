# apitrace

`apitrace` is a graphics API tracing and replay tool for D3D11 / D3D12
translation-layer development.

It captures Windows D3D calls into a directory-based trace bundle and replays
that bundle through `retrace`. The project also provides a Metal trace /
retrace path for debugging translated Metal command streams from downstream
translation layers.
